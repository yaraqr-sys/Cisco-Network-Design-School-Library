# University Campus LAN Design

## Project Overview
Designed and implemented a multi-floor university campus LAN using Cisco Packet Tracer.

## Objectives
- Create VLAN segmentation
- Configure inter-VLAN routing
- Implement IP addressing
- Configure DNS and HTTP server
- Enable VoIP communication

## Technologies Used
- VLAN
- Inter-VLAN Routing
- Static Routing
- IP Addressing
- DNS
- HTTP Server
- Cisco Packet Tracer

## Key Implementations
- Configured 4 VLANs
- Connected 3 routers
- Implemented DNS & HTTP services
- Verified full connectivity using ping tests
